# Hackazone-Security-Assessment-Project
📌 Project Overview
The goal of this project was to identify, exploit, and remediate critical security vulnerabilities following the OWASP Top 10 standards and the CIA Triad (Confidentiality, Integrity, and Availability).

📁 Weekly Reports Summary
[Week 01: Vulnerability Discovery]
Focus: Automated and manual penetration testing.

Key Findings: Identified SQL Injection, Command Injection, and Stored XSS.

Tools Used: OWASP ZAP, Nmap, Burp Suite.

Deliverable: Week_01/Security_Assessment_Report_W1.pdf

[Week 02: Mitigation & Hardening]
Focus: Implementing technical defenses.

Measures Taken: Configured JWT authentication, RBAC (Role-Based Access Control), and Helmet.js security headers.

Deliverable: Week_02/Vulnerability_Mitigation_Report_W2.pdf

[Week 03: Auditing & Final Reporting]
Focus: Post-remediation testing and logging setup.

Implementation: Established a secure logging system using Winston and verified fixes.

Deliverable: Week_03/Final_Security_Audit_Report_W3.pdf

🛠 Tools & Technologies
Testing: OWASP ZAP, Burp Suite, ffuf, Nmap.

Defense: Node.js, Express, JWT, Helmet.js, Winston.
